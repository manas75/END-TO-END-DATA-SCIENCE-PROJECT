## This project demonstrates a complete end-to-end data science workflow, starting from a raw dataset and ending with a deployed machine learning model that can make real-time predictions through an API. The purpose of this project is to showcase how data science is applied in real-world scenarios, where building a model is only one part of the process. The project emphasizes proper data handling, model training, reproducibility, and deployment, making it suitable for internship submissions and beginner portfolios.The project is intentionally designed to be simple and beginner-friendly while still following industry-standard practices. All files in this repository are created using basic tools such as Notepad and are structured clearly so that anyone cloning the repository can understand and run the project without confusion.
